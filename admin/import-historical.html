Historical Calibration Import

⚠ Run this once, then delete this page.
This imports 789 historical calibration records into Firestore. It does not check for duplicates — running it twice will create 1,578 records instead of 789. Once the import succeeds and you've verified the data in the app, delete admin/import-historical.html, admin/historical-data.json, and admin/import-historical.js from the repo.

Before you click import

This writes directly to your live calibrations collection, using the same rules as the app itself (no special access — if the current Firestore rules ever get more restrictive, this may need updating too). Historical scores and Pass/Fail results are preserved exactly as originally recorded, including any past manual scoring errors — nothing is recalculated.
