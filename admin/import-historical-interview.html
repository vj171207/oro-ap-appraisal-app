Historical AP Interview Import

⚠ Run this once, then delete this page.
This imports 159 historical interview records into Firestore. It does not check for duplicates — running it twice will create 318 records instead of 159. Once the import succeeds and you've verified the data in the app, delete admin/import-historical-interview.html, admin/historical-interview-data.json, and admin/import-historical-interview.js from the repo.

Before you click import

This writes directly to your live interview_entries collection, using the same rules as the app itself. Round 1 Decision text and scores are preserved exactly as originally recorded — nothing is recalculated. Note: for many older rows, the "Round 1 Interviewer" field in the source sheet actually contains a remark (e.g. "Ok for APT") rather than an interviewer's name — that's imported as-is, matching what was in the original column.
